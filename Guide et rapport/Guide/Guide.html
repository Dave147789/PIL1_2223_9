    Page d'inscription

    La page d'inscription permet aux utilisateurs de créer un compte sur le site. Les utilisateurs doivent fournir leurs informations personnelles et choisir une licence. Navigation :
    La barre de navigation en haut de la page contient les liens suivants :
    Accueil : Permet de revenir à la page d'accueil du site.
    Contactez-nous : Redirige vers la page de contact.
    Espace Étudiant : Redirige vers la page de connexion pour les étudiants.
    Inscription :
    Remplissez le formulaire d'inscription pour créer un nouveau compte.
    Les champs obligatoires sont les suivants :
    Nom : Entrez votre nom.
    Prénoms : Entrez vos prénoms.
    Email : Entrez votre adresse e-mail.
    Phone : Entrez votre numéro de téléphone.
    Matricule : Entrez votre matricule.
    Licence : Sélectionnez votre licence dans la liste déroulante.
    Mot de passe : Choisissez un mot de passe.
    Confirmer le mot de passe : Retapez votre mot de passe pour confirmation.
    Cliquez sur le bouton "Inscrivez-vous" pour soumettre le formulaire.
    Erreurs :
    Si des erreurs se produisent lors de la soumission du formulaire, un message d'erreur s'affiche. Veuillez vérifier les erreurs signalées et corriger les champs correspondants. Pied de page :
